## About-the-project

这个项目主要以学习为目的，UI参考了QQ音乐.

API接口 :  [NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)

**功能实现：**

​	✅ 登录

​	✅ 播放，切换音乐

​	✅ 播放列表

​	✅ 搜索音乐

​	✅ 歌词页
